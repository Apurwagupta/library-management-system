# Library Management System\n\n## Project Structure\n\n- **library-management-system/**\n  - **backend/**\n    - **config/**  (Configuration files)\n    - **controllers/**  (Business logic)\n    - **models/**  (Database models)\n    - **routes/**  (API routes)\n    - **middlewares/**  (Middleware functions)\n    - **utilities/**  (Helper functions)\n    - **package.json**  (Node.js dependencies)\n    - **server.js**  (Main server file)\n  - **frontend/**\n    - **public/**  (Static files)\n    - **src/**  (React components)\n      - **components/**  (Reusable components)\n      - **pages/**  (Main pages)\n      - **App.js**  (Main app file)\n      - **index.js**  (Entry point)\n    - **package.json**  (React dependencies)\n  - **database/**\n    - **schema.sql**  (Database schema file)\n  - **README.md**  (Project documentation)\n\n## Database Schema\n\n```sql\nCREATE TABLE users (\n    id SERIAL PRIMARY KEY,\n    username VARCHAR(255) NOT NULL UNIQUE,\n    password VARCHAR(255) NOT NULL,\n    email VARCHAR(255) NOT NULL UNIQUE,\n    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP\n);\n\nCREATE TABLE books (\n    id SERIAL PRIMARY KEY,\n    title VARCHAR(255) NOT NULL,\n    author VARCHAR(255) NOT NULL,\n    genre VARCHAR(255),\n    published_date DATE,\n    status VARCHAR(50) DEFAULT 'available'\n);\n\nCREATE TABLE loans (\n    id SERIAL PRIMARY KEY,\n    user_id INT REFERENCES users(id) ON DELETE CASCADE,\n    book_id INT REFERENCES books(id) ON DELETE CASCADE,\n    loan_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,\n    return_date TIMESTAMP\n);\n```\n\n## Backend APIs\n\n- **GET /api/books**: Retrieve a list of all books\n- **POST /api/books**: Add a new book\n- **GET /api/users**: Retrieve a list of all users\n- **POST /api/users**: Register a new user\n- **POST /api/loans**: Create a new loan\n- **GET /api/loans**: Retrieve a list of all loans
